HTML

HTML (Hypertext Markup Language) is the code that you use to create the structure of a web page. It tells the browser how to display the text, images, and other elements on the page.

Here's an example of a simple HTML document:

<!DOCTYPE html>
<html>
<head>
<title>My Web Page</title>
</head>
<body>

Welcome to My Web Page

<p>This is my first web page.</p>
</body>
</html>

To learn more about HTML, check out the following resources:

  • W3Schools HTML Tutorial
  • Mozilla Developer Network HTML Tutorial
  • Codecademy Learn HTML

CSS

CSS (Cascading Style Sheets) is the code that you use to style the elements on your web page. It allows you to control the appearance of the text, images, and other elements on the page.

Here's an example of a simple CSS document:

body {
font-family: Arial, sans-serif;
}

h1 {
font-size: 2em;
}

p {
font-size: 1em;
}

To learn more about CSS, check out the following resources:

  • W3Schools CSS Tutorial
  • Mozilla Developer Network CSS Tutorial
  • Codecademy Learn CSS

JavaScript

JavaScript is a programming language that you can use to add interactivity to your web pages. It allows you to create things like slideshows, animations, and games.

Here's an example of a simple JavaScript script:

<script>
function sayHello() {
alert("Hello, world!");
}
</script>

To learn more about JavaScript, check out the following resources:

  • W3Schools JavaScript Tutorial
  • Mozilla Developer Network JavaScript Tutorial
  • Codecademy Learn JavaScript
